# National Rail â€” Sales Performance Dashboard

A beginner-friendly, object-oriented Streamlit dashboard for exploring railway sales and operational data. The app loads a CSV (or uses a sample dataset), cleans and prepares the data, and provides interactive visualizations for:

- Identifying most popular routes (by trips and revenue)
- Detecting peak travel times (manual and auto-detection)
- Analyzing revenue by ticket type and railcard
- Diagnosing on-time performance and delay contributors
- Refund and delay-group analysis

## Quick start


# install dependencies
pip install streamlit pandas matplotlib seaborn numpy

# run the app
streamlit run app.py
